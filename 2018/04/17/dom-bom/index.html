DOM/BOM

Posted on 2018-04-17

DOM 文档对象模型
attributes: 属性节点
nodeName: 节点名称
nodeType: 节点类型
标签 1 - 有tagName
属性 2
文本 3
注释 8
文档 9

使用JS操作DOM
parentNode, childNodes, previousElementSibling, nextElementSibling, firstElementChild, lastElementChild,
children 所有子元素集合
offsetParent 距离最近的有定位的父级(不为static)
offsetLeft, offsetTop: 没有单位, 不带边框, 父级有定位, 清掉默认样式。
getBoundingClientRect: 没有单位, 不包含滚动条
width 宽度(包含边框)
height 高度(包含边框)
left 从元素的最左边到可视区的最左边距离
right 从元素的最右边到可视区的最左边的距离
top 从元素的最上边到可视区的最上边的距离
bottom 从元素的最下边到可视区的最上边的距离
元素宽高
元素.offsetWidth
元素.clientWidth // 不含边框
元素.offsetHeight
元素.clientHeight // 不含边框
getAttribute(属性名), box.属性, box['属性名']。
setAttribute(attr,value);
removeAttribute(attr);
节点
createElement(tagName);
appenChild(child);
insertBefore();
removeChild();
replaceChild();
要克隆的节点.cloneNode(boolean)
true 包含子节点, false 不包含子节点
不能克隆元素上绑定的js
动态和静态获取元素问题
getElementsByTagName(TagName) 动态获取, 原来元素有变化会重新监听。
querySelectorAll 只获取一次。
DOM操作表单
值 value, input, select, textarea
用name属性获取元素
selectedIndex 下拉框索引
oninput 监听输入
onchange 下拉框内容有变化触发
onsubmit 点击按钮提交
onreset 表单重置触发

BOM 浏览器对象模型
window.open() 打开窗口
window.close() 关闭窗口, 只能关闭window.open打开的窗口
window.navigator.userAgent 用户浏览器, 操作系统等信息
window.location 地址栏信息
window.location.search 地址栏查询信息
window.location.hash 锚点信息
window.innerWidth/window.innerHeight 不兼容低版本浏览器
window.pageXOffset, window.pageYOffset 滚动条距离
window.scrollTop(x,y) 参数都出现
scrollHeight 被内容撑开的高度, 内容没有外层高就是外层的高度。
window.onscroll 滚动条滚动时触发
window.onresize 窗口改变大小时触发
